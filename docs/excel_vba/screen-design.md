# Excel VBA 暫定ツール 画面設計書

**バージョン**: 1.0  
**作成日**: 2026-05-01

---

## 1. 画面の方針

| 種別 | 役割 | 初期リリース |
|---|---|---|
| **シート画面** | 定型用紙・申請リスト等の主たる業務操作面 | ✅ |
| **UserForm（必須）** | パスワード入力・コメント入力等の最低限のダイアログ | ✅ |
| **カード入力 UserForm** | 直感的な申請入力画面 | ✕（**設計のみ**） |
| **環境設定者メニュー** | マスタ編集・パスワード変更等のメニュー | ✅（簡易版） |

```
│                                                                    │

│  申請者   : [山田 太郎 ▼]  所属  : [営業部     ]                   │
├────────────────────────────────────────────────────────────────────┤
│  休暇区分 : [年次有給休暇 ▼]                                        │
│  取得単位 : ◎時間取得  ○日取得（全休/午前/午後）                   │
│  取得日   : [2026-05-10] ～ [2026-05-10]                           │

│  休暇理由 : [_____________________________________________]        │
│  備考     : [_____________________________________________]        │
├────────────────────────────────────────────────────────────────────┤
│  ┌────────┬────────┬────────┬────────┐                              │
│  │ 合議   │ 承認   │ 決裁   │ 完了   │  （決裁フロー欄）             │
│  │        │        │        │        │                              │
│  │  ㊞    │  ㊞    │  ㊞    │  ㊞    │                              │
│  │ 日付   │ 日付   │ 日付   │ 日付   │                              │
│  └────────┴────────┴────────┴────────┘                              │
│                                                                    │
│   現在の休暇残: [自動表示]                                          │
│                                                                    │
│   [ 一時保存 ]  [ 提出する ]  [ 印刷 ]  [ クリア ]                  │
└────────────────────────────────────────────────────────────────────┘
```

#### 入力欄

| セル | 項目 | データ検証 |
|---|---|---|
| C3 | 申請番号 | 提出時に自動採番（手動編集不可） |
| F3 | 申請日 | 自動（編集不可） |
| C4 | 申請者 | ユーザーマスタからのドロップダウン |
| F4 | 所属 | 申請者選択時に自動表示 |
| C6 | 休暇区分 | 「年次有給休暇」固定（将来拡張） |
| C7 | 取得単位 | オプションボタン |
| C8, F8 | 取得日 | 日付入力 |
| C9, F9 | 取得時刻 | `00:00`, `00:15`, …の `15分刻み` リスト |
| C12 | 休暇理由 | 自由入力（任意） |
| C13 | 備考 | 自由入力（任意） |

#### 数式・自動計算

- `取得時間` (=I9) : `=IF(C7="hour",(F9-C9)*24*60,0)` 分
- `取得日数` (=I10) : `=IF(C7="day", DATEDIF(C8,F8,"d")+1, I9/465)`
- `休暇残` : VBA 経由で算出（M_LeaveBalance.GetRemaining）

#### 保護設定

- **編集可能セル**: 申請者、休暇区分、取得単位、取得日、取得時刻、理由、備考
- **保護セル**: タイトル、申請番号、申請日、決裁フロー欄、自動計算欄
- シート保護: 共通保護PW

### 2.2 `申請リスト` シート

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ [▼ステータスフィルタ] [▼期間フィルタ]  [選択行操作: 承認|差戻|決裁|取消|クローズ|印刷] │
├─────────────────────────────────────────────────────────────────────────────┤
│ # │ 申請番号    │ 申請者   │ 休暇区分 │ 取得日       │ 状態        │ 申請日   │
│ 1 │ 2026-05-0001│ 山田 太郎│ 年次有給 │ 5/10 09-11   │ 承認待ち(2) │ 5/01     │
│ 2 │ 2026-05-0002│ 佐藤 花子│ 年次有給 │ 5/12 全休    │ 決裁済     │ 5/02     │
│ 3 │ 2026-05-0003│ 鈴木 一郎│ 年次有給 │ 5/15 午前    │ 完了        │ 5/03     │
│...│             │          │          │              │            │          │
└─────────────────────────────────────────────────────────────────────────────┘
```

- 1行目: テーブルヘッダ（オートフィルタ・並び替え可）
- 上部にコマンドボタン群（`M_Main` のハンドラを起動）
- ソフトデリート行（`is_deleted=TRUE`）はオートフィルタで非表示が既定

#### 行選択 → ボタン操作の流れ

```
1. 操作対象の行をクリック（ActiveCell.Row 取得）
2. ボタン押下
3. VBA: 選択行の application_id を取り出す
4. M_Workflow.* を呼び出し
```

## 3. UserForm 設計

### 3.1 `F_PasswordPrompt`

```
┌──────────────────────────────────┐
│  パスワード入力                  │
├──────────────────────────────────┤
│                                  │
│   [ラベル: caption テキスト]      │
│                                  │
│   ┌──────────────────────────┐   │
│   │ ●●●●●●●●●               │   │
│   └──────────────────────────┘   │
│                                  │
│   [ OK ]    [ キャンセル ]       │
└──────────────────────────────────┘
```

- TextBox の `PasswordChar = "●"`
- Enter キーで OK 動作
- 戻り値: 入力された平文文字列（キャンセル時は空文字列）

```vb
' F_PasswordPrompt
Public Function Prompt(caption As String) As String
    Me.lblCaption.Caption = caption
    Me.txtPassword.PasswordChar = "●"
    Me.Show vbModal
    Prompt = m_result
    Unload Me
End Function

Private Sub btnOk_Click()
    m_result = Me.txtPassword.Text
    Me.Hide
End Sub

Private Sub btnCancel_Click()
    m_result = ""
    Me.Hide
End Sub
```

#### 検証ルール

| 項目 | ルール | エラーメッセージ |
|---|---|---|
| 申請者 | 必須 | 「申請者を選択してください」 |
| 取得日 | From ≤ To | 「取得開始日は終了日以前にしてください」 |
| 取得日（時間取得時） | From == To | 「時間取得は同日のみ可能です」 |
| 取得時刻 | 15分単位 | 「取得時刻は15分単位で指定してください」 |
| 取得時刻 | From < To | 「開始時刻は終了時刻より前にしてください」 |
| コメント（差戻時） | 必須 | 「差し戻しコメントは必須です」 |

#### 状態（UserForm 内部）

```vb
Private m_mode As String     ' "new" / "edit"
Private m_applicationId As String
Private m_resultId As String

Public Function ShowNew() As String
    m_mode = "new"
    Me.Show vbModal
    ShowNew = m_resultId
End Function

Public Function ShowEdit(appId As String) As String
    m_mode = "edit"
    m_applicationId = appId
    LoadFromSheet appId
    Me.Show vbModal
    ShowEdit = m_resultId
End Function
```

## 5. アクセシビリティ・操作性

- すべての入力欄に **タブ順序（TabIndex）** を設定
- ショートカットキー（既定割り当て）:
  - `Ctrl+Shift+S` : 申請提出（フォーム上）
  - `Ctrl+Shift+A` : 選択行を承認
  - `Ctrl+Shift+R` : 選択行を差し戻し
  - `Ctrl+Shift+D` : 選択行を決裁
- 色弱配慮: 状態色は赤緑単独で判別させない（テキストラベル併記）

## 6. 印刷レイアウト

### 6.1 定型用紙（A4縦）

- 印刷範囲: `A1:H35` （実装時調整）
- 余白: 上下 15mm、左右 12mm
- フッタ: 「承認決裁システム（暫定版）  出力日時: &D &T  ページ &P/&N」

### 6.2 一覧表（A4横）

- 印刷範囲: `申請リスト!A1:K{lastRow}`
- 1ページ目に印刷タイトル行を再表示（`PageSetup.PrintTitleRows`）
- 余白: 上下 12mm、左右 10mm

## 7. UI ガイドライン

| 項目 | 設計値 |
|---|---|
| フォント | メイリオ 10pt（既定）、見出しは 12pt 太字 |
| 配色 | ヘッダ: #1F4E78（紺）、背景: #F2F2F2 / 入力欄: 白 |
| ボタン色 | 通常: 標準、決裁系: オレンジ、差戻系: 赤、クローズ系: 緑 |
| エラー表示 | セル背景を薄赤、メッセージは MsgBox |
